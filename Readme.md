# 🤖 AI Chat Assistant (MERN + Gemini API)

An intelligent AI Chat Assistant built using the **MERN Stack** and powered by the **Google Gemini API**. This application provides a modern ChatGPT-like chatting experience with a clean UI, real-time AI responses, and secure backend integration.

---

## 🚀 Features

- 💬 ChatGPT-like conversational interface
- 🤖 AI-powered responses using Google Gemini API
- ⚡ Real-time message exchange
- 🎨 Modern and responsive UI
- 🔒 Secure API key handling on the backend
- 📜 Chat history support
- 🌙 Clean and user-friendly design
- 📱 Mobile responsive
- 🔄 Fast API communication
- 📦 MERN Stack architecture

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/AI-Chat-Assistant.git

cd AI-Chat-Assistant
```

---

### Install Frontend

```bash
cd client
npm install
```

---

### Install Backend

```bash
cd ../server
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

GEMINI_API_KEY=your_google_gemini_api_key
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd server
npm run dev
```

---

### Start Frontend

```bash
cd client
npm run dev
```

The application will be available at

```
Frontend : http://localhost:5173

Backend : http://localhost:5000
```
